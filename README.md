
Simple Stock Analyzer

What is this Simple Stock Analyzer?

This is a simple web app that helps people understand whether a stock looks healthy, average, or risky using basic financial information.

It is made mainly for:

 - Beginners in the stock market

This is a learning tool, not a tip-giving app.



 What can it do?

 Manual mode – enter values and learn what they mean
 Live mode – type a stock name (like `TCS.NS`) and fetch real data
 Explains results in easy, everyday language
 Shows a clear Good / Average / Risky outcome with reasons



Why this app?

Many stock apps show numbers but don’t explain them.
This app tries to answer simple questions like:

 - Is this company actually earning well?
 - Am I paying too much for this share?
 - Is the company risky?



How to run it (quick)

1. Install Python
2. Install libraries


pip install streamlit yfinance pandas matplotlib


3. Run the app


streamlit run stock_analyzer_dual_mode.py
